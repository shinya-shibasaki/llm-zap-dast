# 診断対象マップ — <run-id>

> 根拠の凡例：`[S]` ソース由来 ・ `[I]` 推測。両者は分けて記載する。

## サマリ

- フレームワーク / 起動方法：<例：Express 4、`npm run dev`、ポート3000> `[S]`
- ベースURL：<http://localhost:3000> `[S]`
- 認証モデル：<セッションCookie / JWT / なし> `[S/I]`
